The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM is an object-oriented representation of a web page,which means that everything on the page is represented as an object.

When web-page is loaded ,the browser creates the document object model of the page.
